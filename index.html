Ride Rate
Ride Rate — приложение для оценки поездок на автобусе, а также оценка водитилей и маршрутов. В приложении Ride Rate можно оплачивать поездку, а также выполнять задания за которые будут начисляться балы.

Goly

Goly — приложения для создания и выполнения задач.

Goly
Goly — приложения которое поможет быть более организованным и напомнит о невыполненых задачах. Оно также Геймифецировано так что пользователи могут получать баллы за достигнутые задачи, а потом тратить их на игру

Teleport

Teleport - Приложение с помощью которого можно передвигатся.

Teleport
Teleport - Приложение с помощью которого вашв жизнь станет в разы легче. Это приложение дает возможность людям телепортироватся в любое место выбранное на карте пользователем в приложении. Доступно только для пользователей достигших 18ти лет!
